### Prerequisites

In order to run this project you need:
```sh
 gem install rails
```

### Setup

Clone this repository to your desired folder:

```sh
  cd folder
  git clone git@github.com:DiegoMero/blog.git
```

### Install

Install this project with:

```sh
  cd blog
  bundle install
```
